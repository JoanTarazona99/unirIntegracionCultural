🏗️ ARQUITECTURA ENTREGADA

HYBRID RAG PIPELINE:
┌─ Query Input (ES/EN/RU)
│
├─ Language Detector
│  └─ Identifica idioma → Selecciona modelos específicos
│
├─ Stage 1: BM25 Retrieval
│  └─ Búsqueda rápida por keywords → Top 100 resultados
│
├─ Stage 2: Dense Embeddings
│  └─ Búsqueda semántica → Top 5-10 resultados relevantes
│
├─ Stage 3: Reciprocal Rank Fusion
│  └─ Combina rankings de BM25 + Dense → Top 20 candidatos
│
├─ Stage 4: Cross-Encoder Reranking
│  └─ Fine-grained scoring → Rerank top K*multiplier
│
├─ Stage 5: Score Fusion
│  └─ Weighted combination: (0.3*BM25 + 0.4*Dense + 0.3*Rerank)
│
└─ Output: Top K resultados ordenados
   └─ Format: {id, source, title, content, score, language_detected}

CARACTERÍSTICAS:
├─ Graceful Degradation: Si dense falla → BM25+Rerank, si rerank falla → BM25+Dense
├─ Multiidioma: ES, EN, RU con modelos específicos
├─ Configurable: Pesos, modelos, top_k por ambiente
└─ Backward Compatible: Puede reemplazar search() existente
